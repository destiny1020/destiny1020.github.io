[Effective JavaScript] Item 19 使用高阶函数 (High-Order Function)

发表于 2014-09-15 | 分类于 编程语言 ， JavaScript

本系列作为Effective JavaScript的读书笔记。

重点

不要被高阶函数这个名字给唬住了。实际上，高阶函数只是代表了两类函数：

  1. 接受其他函数作为参数的函数
  2. 返回值为函数的函数

有了这个定义，你也许就发现你已经使用过它们了，典型的就是对于一些事件的处理时传入的回调函数。
