总结

  1. 注意当抽取其他对象的方法并调用的时候，this的指向也许不正确
  2. 当将一个对象的方法传入到高阶函数中时，使用匿名函数来保证在调用该方法时，this的指向正确
  3. 使用bind方法来“修饰”一个方法来确保this的指向是正确的
